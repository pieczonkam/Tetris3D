# Tetris3D

Tetris 3D game made in C++ with SFML, ImGui and OpenGL 2.0

## Controls
- ***A/D***: move block left/right
- ***S***: make block fall faster
- ***Q/W/E***: rotate block around X/Y/Z axis
- ***<-/->***: rotate frame left/right
- ***P***: pause game
- ***Esc***: quit game
